# node_penduduk
### 1. Clone
```
git clone https://github.com/rafirh/node_penduduk.git
```
### 2. Change Directory
```
cd node_penduduk
```
### 3. Init
```
npm init -y
```
### 4. Install 
```
npm install express mysql body-parser
```
### 5. Import sql
```
import above sql file in phpmyadmin
```
### 6. Deploy project
```
node index.js
```
